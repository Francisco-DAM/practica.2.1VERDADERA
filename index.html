Navegadores Web

Un navegador web (en ingés, web browser) es un software, aplicación o programa que permite al acceso a la Web, interpretando la información de distintos tipos de archivo y sitios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegador web

Safari: Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para MacOS, iPadOS e iOS.

Mozilla Firefox: es un navegador web libre y de código abierto desarrollado para Linux, Android, iOS, macOS y Microsoft Windows coordinado por la corporación Mozilla y la Fundación Mozilla.

Google Chrome: Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto.
